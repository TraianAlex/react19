# React 19+ Features Demo Apps

This folder contains small applications demonstrating React 19+ new features:

## Apps Included:

1. **Actions Demo** - Server Actions and form handling
2. **Use Hook Demo** - New `use()` hook for promises and context
3. **Optimistic Updates** - useOptimistic hook demonstration
4. **Form Actions** - Enhanced form handling with actions
5. **Suspense Improvements** - Better Suspense boundaries and error handling
6. **Concurrent Features** - useTransition and concurrent rendering
7. **Asset Loading** - New asset loading capabilities

Each app is self-contained and demonstrates specific React 19 features with practical examples.

## APIs Used:

- JSONPlaceholder (https://jsonplaceholder.typicode.com/)
- Local json-server (localhost:4000)
- Mock data for demonstrations

## Styling:

- Bootstrap 5 classes for consistent styling
- Minimal custom CSS when needed
